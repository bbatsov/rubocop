RuboCop

An experimental Ruby code analyzer, based on the community Ruby style guide.

View on GitHub

Community

RuboCop is a Ruby code style checker based on the community-driven Ruby Style Guide.

Support

RuboCop currently supports MRI 1.9 and MRI 2.0.

Install

RuboCop's installation is pretty standard:

$ gem install rubocop

Usage

Running rubocop with no arguments will check all Ruby source files in the current folder:

$ rubocop

Alternatively you can rubocop a list of files and folders to check:

$ rubocop app spec lib/something.rb

Use -h for available command-line options, including --emacs and --silent.
